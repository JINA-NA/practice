js 조건문 이해하기

조건문의 정의: 어떠한 조건이 부여되고, 참/거짓에 맞는 상황에 따라 선택 도는 행위를 취하는 것

  • 삼항 연산자: 조건문은 아니지만 두 각지의 조건 상황에 맞게 연산하게 되는 항 -> 세가지 상황을 만든다.
  • 단수 조건문(=단수 분기): if/else if -> 하나의 조건을 보고 그것이 참이면 수행하고, 거짓이면 다른 것을 수행하거나 내용을 끝내는 기능. 하난의 기능만을 수행하게 하는 형태이다. 여러 개의 조건 중에서 하나의 선택만 할 수 있다.
  • 다수 조건문(=다수 분기): switch라는 기능을 사용한다. 하나의 조건을 보고 그것이 참에 해당하는 경우에 수행할 수 있는 경우의 수를 모두 행하도록 하는 기능이다.
    단, 경우의 상황이 끝나는 경우에는 반드시 break 구문을 반드싯 작성해야 한다. break 안 하면 뒤의 것까지 다 수행이 되어버린다.
