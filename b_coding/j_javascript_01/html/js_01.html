javascript 문서 사용하는 위치 파악하기

javascript는 제어의 기능을 하는 언어 : html/css문서의 내용을 삽입/삭제/수정
또는 서버를 구성하여 처리하기, 동기화처리, 애니메이션/버튼 클릭시 처리되는 기능

javascript는 java와는 전혀 다른 언어이다. 기능, 형태 모든 게 다르다. 그러니 javascript를 줄여서 java라고 말하지 말 것.

다양한 버전과 다양한 스크립트 언어들이 존재한다.
가장 대표적인 종류는 javascript, coffeescript(지금은 소멸), actionscript(지금은 소멸), typescript, dart(구글에서 제작), ecmascript(표준으로 선정되기 위해)
버전: ecma(줄여서 es)를 표준 기반으로 다양한 버전이 존재하지만 기준은 es5/es6 이상
es5는 익스플로러, es6 그 이후

javascript 문서 사용하는 위치 파악하기

  1. head내부에 작성
  2. body내부에 문서 중간에 작성
  3. body내부 끝에 작성
  4. 외부에서 불러오는 방법(head에서 작성)
  5. 외부에서 불러오는 방법(body에서 작성) <- 주로 이 방식으로 구성한다.
