table 구조 내용 이해하기

table 요소는 과거 웹페이지의 구조를 구성하는 데 필요한 요소였다.과거에!
하지만 현재 웹페이지의 기본은 의미의 전달에 중점을 두고 있기에 table 구조는 사용하지 않는다.
테이블 자체를 사용하지 않는다는 게 아니라, 웹페이지 구조를 짤 때 쓰지 않는다는 것.

table의 기능은 표형식으로 구성 되어있기 때문에, 주식, 경기표, 날씨표, 달력 등에 사용될 수 있다.

의미 전달을 놓고 봤을 때, 사용을 자제해야 하는 부분은 게시판(선생님 왈 list로 짜는 것을 권장)
